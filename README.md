# SFExpress
  模拟顺丰优选网上购物商城

## 技术上的实现
 1. 框架结构：springmvc+spring+mybatis
 2. 服务器：tomcat
 3. 数据库：mysql
 4. 项目管理：maven
 5. 前端页面渲染：thymeleaf
 6. 权限：shiro
